# technogiq-discourse-invite-manager

Adds expiration dates and dynamic metadata to Discourse invitation links.

## Features
- Expiration date for user accounts created from invites
- Dynamic key-value metadata storage per invite
- Auto-deactivation when expired
- Admin API for invite creation with metadata
